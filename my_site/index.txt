1.Установка Django: pip install django
2. Создание нового проекта: django-admin startproject myproject
(Замените myproject на имя вашего проекта)
3.Создание нового приложения: python manage.py startapp myapp

5.Миграции базы данных: Перед запуском приложения, выполните миграции для создания необходимых таблиц в базе данных:
python manage.py migrate
6. Запуск сервера: После того как вы настроили проект, можно запустить сервер разработки:
python manage.py runserver

Сервер будет запущен по умолчанию на http://127.0.0.1:8000/.

Теперь вы можете открыть браузер и перейти по адресу http://127.0.0.1:8000/, чтобы увидеть ваш работающий Django проект.

python manage.py makemigrations
python manage.py migrate
